# i18n (ICU4J + lang/) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all hardcoded strings in 16 Thymeleaf templates with `#{}` message expressions backed by ICU4J-powered EN/FR message files under `lang/`.

**Architecture:** A custom `ResourceBundleMessageSource` bean wraps ICU4J's `MessageFormat` inside a `java.text.MessageFormat` subclass so Spring's existing resolution pipeline is preserved. `useCodeAsDefaultMessage=true` makes missing keys display the key name rather than throw. Message files live in `src/main/resources/lang/`.

**Tech Stack:** Java 17, Spring Boot 4.0 (Spring Framework 7), Thymeleaf 3, ICU4J 76.1, JUnit 5 / AssertJ

- [ ] **Step 5: Create `I18nConfig.java`**

`src/main/java/fr/enimaloc/catapult/config/I18nConfig.java`:
```java
package fr.enimaloc.catapult.config;

import com.ibm.icu.text.MessageFormat;
import org.springframework.context.MessageSource;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.support.ResourceBundleMessageSource;

import java.text.FieldPosition;
import java.util.Locale;

@Configuration
public class I18nConfig {

    @Bean
    public MessageSource messageSource() {
        var source = new ResourceBundleMessageSource() {
            @Override
            protected java.text.MessageFormat createMessageFormat(String msg, Locale locale) {
                return new IcuWrappedMessageFormat(msg, locale);
            }
        };
        source.setBasenames("lang/messages");
        source.setDefaultEncoding("UTF-8");
        source.setUseCodeAsDefaultMessage(true);
        return source;
    }

    /**
     * Wraps ICU4J MessageFormat inside java.text.MessageFormat so Spring's
     * ResourceBundleMessageSource can use it without API changes.
     * ICU handles CLDR plural rules; standard {N} substitution also works.
     */
    static class IcuWrappedMessageFormat extends java.text.MessageFormat {

        private final MessageFormat icu;
        private final String pattern;

        IcuWrappedMessageFormat(String pattern, Locale locale) {
            super("", locale); // empty pattern — parent is only here for type compatibility
            this.pattern = pattern;
            this.icu = new MessageFormat(pattern, locale);
        }

        @Override
        public StringBuffer format(Object[] arguments, StringBuffer result, FieldPosition pos) {
            return result.append(icu.format(arguments));
        }

        @Override
        public String toPattern() {
            return pattern;
        }
    }
}
```
