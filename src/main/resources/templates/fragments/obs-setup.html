OBS — Détection par processus

Génère une clé API pour connecter le script OBS à Catapult.

📋 Guide d'installation
  1. Installe Python et OBS Studio (v28+).
  2. Dans un terminal, installe les dépendances :
    pip install psutil requests
  3. Dans OBS : Outils → Scripts → +, sélectionne le fichier catapult_obs.py ci-dessous.
  4. Le script s'exécute automatiquement — aucune configuration supplémentaire.
